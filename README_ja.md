## 概要

CommuKit は、マルチプロトコルネットワーククライアント（TCP、UDP、MQTT、WebSocket）、プログラマブルメッセージパーサー、プロトコルアナライザ、ネットワークツールスイートを 1 つのワークベンチに統合した macOS ネイティブアプリです。

**無料・広告なし・Apple ネイティブ体験。macOS 14+ 対応。**

## 主な機能

### マルチプロトコル接続
ネイティブプロトコルサポートで任意のエンドポイントに接続。Network フレームワークによる TCP/UDP、完全なセッション制御（QoS、キープアライブ、自動サブスクライブ）を備えた MQTT、カスタムヘッダーと SSL 対応の WebSocket。UTF-8、ASCII、GB2312、GBK、GB18030、Big5、Shift-JIS エンコーディングに対応。

### プログラマブルメッセージパーサー
Parse、Filter、Compute、Compose、Algorithm、Mapping、Script の 7 種類のハンドラを備えたビジュアルメッセージパーサー。hex/JSON/XML/文字列から値を抽出し、条件でフィルタし、JavaScript 式で計算し、構造化された出力に変換します。ステップバイステップのデバッグログと中間データスナップショット付き。

### プロトコルアナライザ
MODBUS RTU/TCP および DL/T 645 フレームの自動検出と解析。生の hex を入力するだけで、スレーブアドレス、ファンクションコード、レジスタ値、CRC-16 検証といった構造化されたフィールド解析が得られます。読み取り/書き込み操作用の MODBUS フレーム構築機能を内蔵。

### ネットワークツールスイート
7 つの必須ツールを 1 箇所に：ICMP Ping、進捗表示付き TCP ポート範囲スキャン、WHOIS ドメイン検索、DNS 正引き/逆引きクエリ、TLS 証明書検査、インタラクティブ Telnet セッション、IP 位置情報。ツールはドラッグ＆ドロップで並べ替え可能。

### セッションとワークスペース管理
カスタムアイコンとカラーで接続をグループ化。各接続は複数の独立したセッションを生成可能。接続、グループ、タブのドラッグ＆ドロップによる並べ替え。アプリ再起動後も完全な状態を保持し、自動バックアップ付き。

### データ処理ユーティリティ
チェックサム計算（CRC-16、Checksum、LRC）、圧縮（GZIP、Deflate、Zlib）、暗号化（Base64、AES ECB/CBC 128/256 ビット）、Hex-データ相互変換を内蔵。等幅フォントの専用メッセージエディタ付き。

## 課題と解決策

| 課題 | 解決策 |
|---|---|
| プロトコルごとに異なるツールが必要 | オールインワンワークベンチ — すべてのプロトコルとツールを 1 つに |
| 手動 hex 解析が煩雑でエラーが発生しやすい | ビジュアルメッセージパーサー — 一度ルールを定義すれば自動抽出 |
| 汎用ツールが産業用プロトコルを理解しない | 内蔵プロトコルアナライザ — MODBUS と DL/T 645 を自動検出 |
| ネットワークユーティリティが散在 | 統合ツールスイート — サイドバーに 7 ツール |
| 再起動で接続状態が失われる | 完全なワークスペース永続化 — 多層自動バックアップ |
| 非 ASCII エンコーディング（GB2312、Shift-JIS 等）で文字化け | 8 種類のエンコーディング対応 — 接続ごとに切り替え |
